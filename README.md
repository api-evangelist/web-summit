# Web Summit

[Web Summit](https://websummit.com/) is one of the world's largest technology conferences, held annually in Lisbon, Portugal each November. Founded in Dublin in 2009 as a 150-person gathering, it has grown to attract over 70,000 attendees from across the globe, including founders, investors, policymakers, and executives from the technology industry. Web Summit is organized by the same company that runs Collision (Toronto), RISE (Hong Kong), Web Summit Rio (Rio de Janeiro), and Web Summit Qatar (Doha).

## Overview

Web Summit connects startups, established technology companies, investors, and media through a combination of keynote stages, breakout tracks, networking sessions, and exhibition floors. The event features dedicated programmes for early-stage startups (ALPHA and BETA), a developer programme offering discounted access for open source contributors, and a competitive PITCH competition for startup teams. Its proprietary event technology platform uses data analytics and network science to facilitate attendee connections and meeting scheduling through the Web Summit mobile app.

## Developer Programme

Web Summit offers a Developer Programme that provides discounted ticket access for active developers and open source contributors. This is not a public API program but rather a recognition programme for members of the open source community. Applications typically open in July each year. More information is available at [websummit.com/developers](https://websummit.com/developers/).

## Startup Programme

Web Summit runs an ALPHA and BETA startup programme open to early-stage companies. Selected startups receive exhibition space, investor meeting access, masterclass eligibility, and entry into the PITCH competition. Applications are reviewed by the Web Summit startups team. Details at [websummit.com/startups](https://websummit.com/startups/).

## Mobile App

The Web Summit mobile app is the primary tool for attendees to browse the schedule, discover other attendees and companies, send connection requests, and schedule one-on-one meetings during the event. Attendees can filter by role, industry, and location, and export their connections post-event. The app is available for iOS and Android.
